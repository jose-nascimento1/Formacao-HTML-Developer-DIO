# HTML Web Developer Training
Repository created for code challenges performed in Training. With a 21-hour track, focused on understanding how to work with HTML, the basis for creating the first WEB pages. Learning in practice how to structure your page's markup and how each tag works, knowing the semantic elements and other good practices when building your first page, mastering the concepts necessary to use HTML.
